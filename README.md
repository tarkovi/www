# WWW

Hi internauts! This is my personal website made with [Astro](https://astro.build/). Here you'll find a bit about me, along with some small experiments and explorations in tech. Feel free to explore, get inspired, or even use it as a base for your own website or portfolio!

## How to run the project?

1. Install all the dependencies:

```bash
pnpm install
```

2. Start the project with the following command:

```bash
pnpm dev
```
